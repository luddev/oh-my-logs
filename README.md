# oh-my-logs
oh-my-logs is a handy node.js utility for reading logs.

# Version
---------------------
0.0.1

# Installation

```sh
$ git clone https://github.com/ludkiller/oh-my-logs.git     
$ cd oh-my-logs     
$ npm install       
$ node index.js     
```

See config.js for minimal configuration.
